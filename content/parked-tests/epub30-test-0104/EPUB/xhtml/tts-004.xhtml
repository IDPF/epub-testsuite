CSS3 Speech

The -epub-speak property

[OPTIONAL] speech-010 none

Tests whether content styled using the none value is omitted from aural rendering.

This paragraph should not be announced.

If no text is announced after the description, the test passes. If the text "this paragraph should not be announced" is rendered, the test fails.

[OPTIONAL] speech-020 normal

Tests whether hidden content is announced when the -epub-speak property is set to normal.

I am a hidden paragraph.

If no text is announced after the description, the test falis. I the text "I am a hidden paragraph" is rendered, the test fails.

The -epub-speak-as property

[OPTIONAL] speech-030 spell

Tests support for speaking out letters in a word or acronym one at a time.

water

The test fails if the the word "water" is not spelled out one character at a time.

[OPTIONAL] speech-040 digits

Tests support for speaking out digits in a number one at a time.

42

If the numbers four and two are announced, the test passes. If the number forty-two is announced, the test fails.

[OPTIONAL] speech-050 literal-punctuation

Tests support for speaking punctuation marks.

for (var i = 0; i += 1; i++) { print 'hello' . ' ' . 'world'; }

If the semicolons, periods, parentheses, braces and quotation marks are announced in the preceding line of code, the test passes.

[OPTIONAL] speech-060 no-punctuation

Tests support for omitting punctuation.

I said: here be dragons; there be The Killer Rabbit of Caerbannog

If the colon is not announced and no pause occurs after the semicolon, the test passes.
