[OPTIONAL] fxl-120 Right-to-left page progression direction

Tests right-to-left page progression direction for fixed layout pages.

If page progression is from the right to the left (e.g., the table of contents is to the right of this page in the synthetic spread), the test passes. If page progression is from left to right (e.g., the table of contents is to the left of this page), the test fails. If no spread is generated, the test still passes if pages advance from the left (e.g., clicking on an arrow on the left-hand side, or swiping from left to right).
