Styling

[REQUIRED] svg-110 letter-spacing

Tests whether the letter-spacing attribute is supported.

If the space between letters is different for each line, the test passes.

[REQUIRED] svg-120 word-spacing

Tests whether the word-spacing attribute is supported.

If the space between words is different for each line, the test passes.

[REQUIRED] svg-130 font-size

Tests whether the font-size attribute is supported.

If the size of the two last lines is different than the first line, the test passes.

[REQUIRED] svg-140 font-weight

Tests whether the font-weight attribute is supported.

If the weight of the second line is different than the first line, the test passes.

[REQUIRED] svg-150 font-style

Tests whether the font-style attribute is supported.

If the font-style of the second line is different than the first line, the test passes.

[REQUIRED] svg-160 text-decoration attribute

Tests whether the text-decoration attribute is supported.

If the second line is underlined and the third has a line through the text, the test passes.

Positioning

[REQUIRED] svg-210 Lines

Tests whether the x and y attributes on text elements for positioning lines is supported.

If the vertical alignment of the three lines is different, the test passes.

[REQUIRED] svg-220 Words and Characters

Tests whether the SVG code for positioning words or characters is supported using the text and tspan elements.

If the text "This is a veeery biiig wave" rises and falls in an arc like a wave, the test passes.

[REQUIRED] svg-230 Text: path element

Tests whether positioning text on a path is supported.

If the text follows a parabolic path, the test passes.

[REQUIRED] svg-240 Text: text element with multiple x and y values

Tests whether multiple values of the x and y attributes are supported.

If the text goes down, the test passes.

Rotating

[REQUIRED] svg-310 rotate

Tests whether the rotate attribute is supported.

If the characters in the preceding sentence are rotated, the test passes.

[REQUIRED] svg-320 transform

Tests whether transform attribute (translate and rotate values) are supported.

If each line of text is rotated by the degree it indicates, the test passes.
