Forms

Elements

[OPTIONAL] forms-010 button

Tests whether the HTML5 button element is supported.

If a button with the label "Test" is rendered before this paragraph, the test passes.

[OPTIONAL] forms-020 select: single option

Tests whether the HTML5 select element is supported for picking a single option.

If a drop-down selection box with the options "lorem", "ipsum" and "dolor" is rendered before this paragraph, the test passes.

[OPTIONAL] forms-030 select: multiple options

Tests whether the HTML5 select element is supported for picking multiple options.

If a selection box allowing any or all of options "lorem", "ipsum" and "dolor" to be selected is rendered before this paragraph, the test passes.

[OPTIONAL] forms-040 optgroup

Tests whether the HTML5 optgroup element is supported for grouping options under a common heading.

If the non-selectable heading "Latin" groups the options "lorem", "ipsum" and "dolor" in the preceding selection box, the test passes.

[OPTIONAL] forms-050 datalist

Tests whether the HTML5 datalist element for predefining options is supported.

If, when activating the preceding text box, the options "lorem", "ipsum" and "dolor" are made available for selection, the test passes. If the three options are visible as text below the selection box, the test fails.

[OPTIONAL] forms-060 textarea

Tests whether the HTML5 textarea element is supported.
