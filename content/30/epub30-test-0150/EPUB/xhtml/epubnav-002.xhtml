toc nav

[REQUIRED] epubnav-010 Access

Tests whether the toc nav can be accessed.

If the Reading System provides a means to access the table of contents, the test passes.

If this test fails, all subsequent tests for the toc-nav should be marked Not Supported.

[REQUIRED] epubnav-020 Document-level linking

Tests whether linking to Content Documents from the toc nav is possible.

To complete this test, first bring up the table of contents. Find the "Test Links" section at the bottom and activate the "Index" link.

If a document with the title "Index" is loaded, the test passes.

[REQUIRED] epubnav-030 Fragment-level linking

Tests whether linking to fragments within Content Documents from the toc nav is possible.

To complete this test, first bring up the table of contents. Find the "Test Links" section at the bottom and activate the "Glossary" link.

If the heading "Glossary" appears on the page that loads, the test passes.
