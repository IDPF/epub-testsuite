Custom nav elements

[REQUIRED] epubnav-090 Access

Tests whether custom nav elements can be accessed.

If the Reading System provides a means to access the custom list of illustrations, the test passes.

If this test fails, all subsequent tests for custom nav elements should be marked Not Supported.

[REQUIRED] epubnav-100 Linking

Tests whether linking to Content Documents from a custom nav is possible.

To complete this test, bring up the list of illustrations and activate the "All Figures" link.

If a document with the title "Figures" is loaded, the test passes.

[REQUIRED] epubnav-110 Fragment-level linking

Tests whether linking to fragments within Content Documents from a custom nav is possible.

To complete this test, bring up the list of illustrations and activate the "Figure 5" link.

If a figure whose caption begins "Figure 5" appears on the page that loads, the test passes.
