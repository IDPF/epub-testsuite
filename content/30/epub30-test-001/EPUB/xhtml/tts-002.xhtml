[OPTIONAL] pls-010 PLS: IPA Support

Tests whether PLS lexicons with IPA pronunciations are supported.

Today

If the preceding word is pronounced "tomorrow", IPA lexicons are supported.

[OPTIONAL] pls-020 PLS: X-SAMPA Support

Tests whether PLS lexicons with X-SAMPA pronunciations are supported.

Daily

If the preceding word is pronounced "weekly", X-SAMPA lexicons are supported.

[OPTIONAL] pls-030 PLS: Mixed Lexicon Support

Tests whether X-SAMPA and IPA pronunciations can be mixed in a single lexicon file.

Despair

If the preceding word is pronounced "joy", mixed lexicons are supported.

[OPTIONAL] pls-040 PLS: Multilingual Support

Tests whether PLS lexicons are supported for embedded languages.

Comparing apples to oranges.

If the preceding sentence is pronounced "comparing apples to apples" the test passes, otherwise it fails.

[OPTIONAL] pls-050 PLS: Case Sensitivity

Tests whether PLS lexicons entries are case sensitive.

Bass not bass.

If the preceding sentence is pronounced "comparing apples to apples" the test passes, otherwise it fails.

Friday

[OPTIONAL] speech-010 CSS3 Speech: -epub-speak none

Tests whether content styled using the none value is omitted from aural rendering.

This paragraph should not be announced.

If no text is announced after the description, the test passes. If the text "this paragraph should not be announced" is rendered, the test fails.

[OPTIONAL] speech-020 CSS3 Speech: -epub-speak normal

Tests whether hidden content is announced when the -epub-speak property is set to normal.

I am a hidden paragraph.

If no text is announced after the description, the test falis. I the text "I am a hidden paragraph" is rendered, the test fails.

[OPTIONAL] speech-030 CSS3 Speech: -epub-speak-as spell

Tests support for speaking out letters in a word or acronym one at a time.

water

The test fails if the the word "water" is not spelled out one character at a time.

[OPTIONAL] speech-040 CSS3 Speech: -epub-speak-as digits

Tests support for speaking out digits in a number one at a time.

42

If the numbers four and two are announced, the test passes. If the number forty-two is announced, the test fails.

[OPTIONAL] speech-050 CSS3 Speech: -epub-speak-as literal-punctuation

Tests support for speaking out digits in a number one at a time.

for (var i = 0; i += 1; i++) { print 'hello' . ' ' . 'world'; }

If the semicolons, periods, parentheses, braces and quotation marks are announced in the preceding line of code, the test passes.

[OPTIONAL] speech-060 CSS3 Speech: -epub-speak-as no-punctuation

Tests support for speaking out digits in a number one at a time.

I said: here be dragons; there be The Killer Rabbit of Caerbannog

If the colon is not announced and no pause occurs after the semicolon, the test passes.
