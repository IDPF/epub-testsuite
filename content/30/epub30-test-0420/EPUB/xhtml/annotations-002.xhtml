[OPTIONAL] anno-040 Export Annotations

Tests whether the reading system allows annotations to be exported.

Annotated?

If the reading system provides an option to export annotations, perform the following steps:

  1. Create an annotation on the preceding paragraph with the text "Annotated?".
  2. Export the annotations.
  3. Validate the exported annotation file against the json-ld schema available at http://www.idpf.org/epub/oa/schema/oa-epub-schema.json. An online validator is available at http://jsonschemalint.com/draft4/.

If all of the above steps can be completed successfully, the test passes.

If this test fails, mark the following test anno-050 as Not Supported.

[OPTIONAL] anno-050 Reimport Annotations

Tests whether exported annotations can be reimported.

Delete the annotation created in the previous step and import the annotation file that was generated. If the annotation reappears, the test passes.
