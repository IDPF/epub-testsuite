[OPTIONAL] flow-010 Paginated Flow

Tests whether setting the global rendition:flow property to paginated results in dynamically paginated pages.

If the content in the following chapters is dynamically paginated by the reading system, the test passes.

Note: Some reading systems allow readers to manually change the flow (e.g., from dynamically paginated to scrolled). Do not use these settings for this test. To pass, the reading system must adapt to the specified flow.
