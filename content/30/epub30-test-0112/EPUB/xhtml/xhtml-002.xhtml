[OPTIONAL] flow-020 Continuously-Scrolled Flow

Tests whether setting the global rendition:flow property to scrolled-continuous results in a continously scrolling interface.

If the content of this publication is preesnted as one continuous scrolling document, the test passes.

Note: Some reading systems allow readers to manually change from the flow (e.g., from dynamically paginated to scrolled). Do not use these settings for this test. To pass, the reading system must adapt to the specified flow.
