Basic Tests

[OPTIONAL] mo-basic-210 Media Overlays Playback

Tests whether playback of Media Overlays is supported.

If there is a play/pause button that starts playing audio narration for the text on this page, then the Reading System supports Media Overlays (MO). Implied in this test is that the Reading System still recognizes MO even when the first chapter of the book does not contain it.

[OPTIONAL] mo-basic-220 Synchronized Text Highlighting

Tests whether text highlighting is supported.

If the matching text is highlighted as MO playback proceeds, then the Reading System supports synchronized text highlighting.

[OPTIONAL] mo-basic-230 Text Highlighting: Authored Styles

Tests whether authors can control the highlighting color.

If the text is highlighted with a yellow background phrase-by-phrase as MO playback proceeds, then the Reading System supports authored styles for text highlighting.

[OPTIONAL] mo-basic-240 Text Highlighting: Custom Styles

Tests whether the a reading system implements its own default styles.

If the text is highlighted with another style than what is mentioned above, then the Reading System has its own default styles for text highlighting.
