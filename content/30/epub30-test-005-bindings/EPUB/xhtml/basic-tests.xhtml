Bindings

bindings-basic Simple Binding

Test whether bindings are supported.

If the reading system is able to call the associated binding page, and scripting is enabled, a PASS message will be displayed.

bindings-defaults Binding with Invoked File Information

Tests whether data and type attributes on the bound object are sent to the binding page in the query string.

If the reading system is able to call the associated binding page, and the invoking file information is available in the query string, a PASS message will be displayed.

bindings-params Binding with Parameters

Tests whether parameters in the object are sent to the binding page in the query string.

If the reading system is able to call the associated binding page, and the embedded param is passed in the query string, a PASS message will be displayed.
