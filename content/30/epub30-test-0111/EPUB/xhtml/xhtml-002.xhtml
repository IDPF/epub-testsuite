[OPTIONAL] flow-015 Scrolled Flow

Tests whether setting the global rendition:flow property to scrolled-doc results in scrollable documents.

If each of the documents in this publication is vertically scrollable, and swiping horizontally moves from one to the next, the test passes.

Note: Some reading systems allow readers to manually change the flow (e.g., from dynamically paginated to scrolled). Do not use these settings for this test. To pass, the reading system must adapt to the specified flow.
