このテストファイルについて：
このテストファイルには、日本語の縦書きに関するテストが含まれます。
日本語の縦書き文書では、ページ進行方向は右から左（rtl）とするのが一般的です。本テストファイルも同様です。
そのため、テスト2以降は、テスト1の page-progression-direction="rtl" テストをパスした場合のみ有効となります。

About this test file:
This test file contains tests for Japanese vertical writing documents.
In Japanese vertical writing documents, page progression direction is typically right-to-left. So is this test file.
Therefore, test 2 and below will be valid only if test 1 (page-progression-direction="rtl") is passed.
