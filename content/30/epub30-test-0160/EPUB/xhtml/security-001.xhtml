Tests whether access (read and write) to the window.parent object context is forbidden, from the iframe (if any) that contains this content document.

Javascript scripting must be supported for this test to execute properly.

The the test passes if this content document is not hosted by an iframe (e.g. native UIWebView), or if the Javascript code executed in this document is unable to access the parent object context, as indicated by the text below ("PASS" or "FAIL").
