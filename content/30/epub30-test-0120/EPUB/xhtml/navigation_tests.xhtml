Tests whether previous/next phrase navigation is supported.

While Media Overlays (MO) are playing, use the phrase navigation commands to move through contiguous audio phrases.

This is a paragraph.

This is a sentence within a paragraph. This sentence is divided into individual words. This is the last sentence of the paragraph.

If previous/next navigation is possible at the granularity of audio phrases, the test passes.
