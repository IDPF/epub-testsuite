[OPTIONAL] mo-audio-080 Path resolution for audio files

Tests whether two audio files with the same name, but in different directories, can be referenced.

In this test, allow the Reading System to continue playback and just make sure the highlighted text matches what you hear.

This test uses a single Media Overlay that, in turn, references two mp3 files. The files have the same name but are in different directories.

If the audio playback matches the text above, the test passes.
